# Studiora

## By: Cocowit Team

A simple and modern productivity web application designed to help students manage their academic life in one place. This platform combines task management, note-taking, and focus tracking to support better organization and consistent study habits.



## Overview

This web app provides a centralized workspace where students can:

- Track and manage assignments
- Organize class notes
- Monitor study progress
- Improve focus using a built-in timer

The interface is clean, intuitive, and visually informative, making it easy to understand your progress at a glance.


## Features
### 1. Dashboard

The Dashboard gives a quick overview of your academic activity and productivity.

It includes:

- Summary cards showing total tasks, completed tasks, focus hours, and study streak
- A weekly study hours chart to track your learning patterns
- A task status chart (To Do, In Progress, Done)
- A weekly calendar with upcoming deadlines
- A list of today’s tasks

This page acts as your control center, helping you stay aware of your workload and progress.



### 2. Task Management

The Task feature allows you to manage assignments efficiently.

You can:

- View tasks grouped by status:
  - To Do  
  - In Progress  
  - Done  
- Add new tasks
- Edit task details
- Delete tasks
- Update task status
- Search tasks easily
- Assign priorities (Low, Medium, High)
- Set deadlines with due date indicators
- Switch between Board view and Calendar view

This structure helps you clearly see what needs to be done and what has already been completed.



### 3. Notes

The Notes feature is designed to help you store and organize lecture materials.

You can:

- Create notes for different subjects
- Search notes quickly
- Add tags for better organization
- Edit and delete notes
- Filter notes by subject

The card-based layout makes your notes easy to read and revisit when studying or preparing for exams.


### 4. Focus Timer

The Focus Timer helps you stay concentrated while studying.

It includes:

- A focus session timer
- A break timer
- Start, stop, and reset controls
- Charts to track focus time and break time

This feature encourages better time management and helps you build consistent study habits.



## Purpose

This web application was built to simplify academic organization for students. By combining task tracking, note management, and focus monitoring into a single platform, it supports productivity without overwhelming the user.

The goal is to create a practical and user-friendly tool that helps students stay organized, meet deadlines, and maintain consistent study routines.
